Sistema de Gerenciamento para Pet Shop 🐾

Sistema desenvolvido em Java (Swing) para gerenciamento completo de um Pet Shop, incluindo cadastro de clientes, pets, serviços e pacotes promocionais, com persistência de dados em arquivos CSV.

📋 Funcionalidades

  • Gestão de Clientes: Cadastro, listagem, busca e exclusão.
  • Gestão de Pets: Cadastro de animais vinculados a um dono específico.
  • Agendamento: Contratação de serviços (Banho, Tosa, etc.) com validação de data futura.
  • Pacotes Promocionais: Criação de pacotes com desconto automático.
  • Dashboard: Painel principal com menu lateral moderno e atalhos rápidos.
  • Persistência: Todos os dados são salvos automaticamente em arquivos .csv.

🚀 Tecnologias Utilizadas

  • Linguagem: Java 11+
  • Interface Gráfica: Java Swing (AWT/Swing)
  • Armazenamento: Arquivos de texto (.csv)
  • IDE: Eclipse

📂 Estrutura do Projeto

  • src/main/java/com/petshop/gui: Telas e interface visual.
  • src/main/java/com/petshop/model: Classes de dados (Cliente, Pet, Servico, Pacote).
  • src/main/java/com/petshop/service: Lógica de negócios e manipulação de arquivos CSV.

⚙️ Como Rodar

  1. Clone este repositório.
  2. Importe o projeto no Eclipse ou sua IDE de preferência.
  3. Execute a classe principal: com.petshop.PetShopApplication.java (ou TelaMenu.java).
  4. O sistema criará automaticamente os arquivos .csv na pasta raiz caso não existam.
